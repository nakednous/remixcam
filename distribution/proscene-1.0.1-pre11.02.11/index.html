proscene

A library by Jean Pierre Charalambos for the programming environment processing. Last update, 03/12/2011.

Proscene (pronounced similar as the Czech word "pros√≠m" which means "please") provides classes to ease the creation of interactive 3D scenes in processing. Proscene has been pretty much inspired in the Qt's OpenGL C++ libqglviewer library and thus it has a very similar functionality and API reference to that found in libqglviewer. Main features include (but are not limited to):

  • 1. Default interactivity to your scenes
  • 2. Arcball, walkthrough and third person camera modes.
  • 3. Hierarchical coordinate systems (frames).
  • 4. Frames can easily be moved with the mouse.
  • 5. Keyframes.
  • 6. Object picking
  • 7. Keyboard shortcuts and camera profiles customization
  • 8. Animation framework
  • 9. Off-screen rendering support



Download

Download proscene version 1.0.1-pre11.02.11 in .zip format.

Installation

Unzip and put the extracted proscene folder into the libraries folder of your processing sketches (or directly into your processing's libraries folder). Reference and examples are included in the proscene folder.

Keywords viewport manipulation, camera modes, frames, keyframes, visibility algortihms

Reference. Have a look at the javadoc reference here. a copy of the reference is included in the .zip as well.

Source. The source code of proscene is available at google code, and its repository can be browsed here.
